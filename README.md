Модель строит рыночно-согласованный прогноз ключевой ставки на основе двух кривых: KRS и OIS.

Сначала берутся MID-котировки KRS по срокам 3M, 6M, 9M, 1Y, 2Y и далее. KRS показывает фиксированную ставку свопа против будущей ключевой ставки. OIS-кривая RUONIA используется для построения дисконт-факторов, которыми приводятся к текущей стоимости как фиксированные, так и плавающие платежи свопа.

Далее создаётся ежедневная шкала дат и календарь заседаний Банка России. До первого будущего заседания сохраняется текущая ключевая ставка. После этого ставка может меняться только в даты заседаний и остаётся постоянной между ними.

Плавающая нога KRS рассчитывается поквартально. Для каждого квартала определяется средняя ключевая ставка, действовавшая в течение квартала с учётом всех заседаний внутри него. Полученный квартальный платёж дисконтируется по OIS-кривой.

Подбор траектории выполняется последовательно по срокам:

3M → 6M → 9M → 1Y → 2Y → …

Сначала подбирается уровень ключевой ставки, при котором рассчитанный KRS 3M совпадает с рыночным MID. Затем, сохраняя уже подобранную короткую часть траектории, подбирается следующий уровень для совпадения с KRS 6M, затем 9M и так далее.

При этом модель не определяет отдельное значение для каждого заседания. Все заседания, относящиеся к одному участку между соседними сроками KRS, получают одинаковый новый уровень ключевой ставки. Если внутри квартала несколько заседаний, квартальный платёж учитывает, сколько дней действовал каждый уровень ставки.

Результат — ступенчатая траектория КС, которая меняется в даты заседаний ЦБ и обеспечивает совпадение рассчитанных KRS с рыночными MID при дисконтировании по OIS.

это рыночная траектория КС, согласованная с ожиданием рынка по видам, в которую могут входить ожидания участников рынка, срочная премия, ликвидность и другие рыночные факторы.


Как рассчитывается спред ЕТС к прогнозной КС
Спред показывает, насколько фиксированная ставка ЕТС на конкретный срок выше или ниже рыночно ожидаемой ключевой ставки за сопоставимый период.
Для расчёта используются:
•	ставка ЕТС;
•	прогнозная траектория КС, восстановленная из KRS;
•	OIS-дисконт-факторы;
•	доли соответствующих периодов.
Средняя прогнозная КС рассчитывается как дисконтированное среднее:
Средняя КС = сумма(KС × OIS DF × доля периода) / сумма(OIS DF × доля периода).
Аналогично фиксированная ЕТС приводится к сопоставимой годовой конвенции:
ЕТС = ТС sum / сумма(OIS DF × доля периода).
После этого:
Спред = ЕТС − средняя прогнозная КС.
Например, если ЕТС на 6 месяцев равна 14,30%, а средняя прогнозная КС — 14,05%, спред составляет 0,25 п.п.
Положительный спред означает, что фиксированная трансфертная ставка выше ожидаемой КС. В нём могут отражаться срочная премия, стоимость ликвидности, риск фиксации ставки, внутренние надбавки и стоимость фондирования. Отрицательный спред означает, что ЕТС ниже ожидаемой КС.
Особенность расчёта в том, что сроки ЕТС заданы в днях, но сопоставляются с месячными бакетами прогноза КС:
•	61 день → 2M;
•	91 день → 3M;
•	122 дня → 4M;
•	151 день → 5M;
•	181 день → 6M;
•	273/274 дня → 9M;
•	365 дней → 1Y.
Поэтому результат близок к простой разнице между ЕТС и средней прогнозной КС, но может немного отличаться из-за того, что точный срок в днях не всегда совпадает с полным календарным месячным горизонтом.
