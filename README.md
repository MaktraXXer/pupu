Вот английские формулы, но с точкой с запятой как разделителем аргументов.

Название бакета:

=TEXT(A2;"0.0%")&" - "&TEXT(A2+0.5%;"0.0%")

Аналог ВПР:

=VLOOKUP(E2;dict_rate_bucket!A:B;2;FALSE)

Если у тебя Excel с английской локалью и разделитель аргументов — запятая, тогда так:

=TEXT(A2,"0.0%")&" - "&TEXT(A2+0.5%,"0.0%")
=VLOOKUP(E2,dict_rate_bucket!A:B,2,FALSE)
