В качестве исходных данных используются два рыночных источника. История ежедневной ставки RUONIA применяется для калибровки параметров модели CIR, описывающих свойства краткосрочной ставки: скорость возврата к среднему, долгосрочный уровень и волатильность. Текущая OIS-кривая RUONIA является основой для построения начальной рыночной структуры ставок: из неё рассчитывается функция ZCYC, а затем детерминированный сдвиг $\varphi(t)$, который заставляет CIR++ согласовываться с наблюдаемой OIS-кривой.

Таким образом, основа CIR++ состоит из двух частей:

$$
r_t=x_t+\varphi(t),
$$

где $x_t$ — случайная CIR-компонента, а $\varphi(t)$ — рыночная часть, заданная текущей OIS-кривой. Волатильность и форма случайных отклонений определяются параметрами, оценёнными по истории RUONIA.

Далее Monte Carlo многократно моделирует месячные переходы CIR-процесса по формуле:

$$ x(t + \Delta t) = x(t) + a (\theta - x(t)) \Delta t + \sigma \sqrt{x(t)} \sqrt{\Delta t} \, Z(t), $$

где $Z(t)$ — случайное стандартное нормальное число, а $\Delta t = 1/12$.

Для каждого сценария генерируется собственная последовательность случайных шоков. Затем к случайной CIR-компоненте $x(t)$ добавляется детерминированный сдвиг $\varphi(t)$:

$$ r(t) = x(t) + \varphi(t). $$

В результате получается множество возможных траекторий будущей краткосрочной ставки, согласованных с текущей OIS-кривой и исторически оценённой динамикой RUONIA.
